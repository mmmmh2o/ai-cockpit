# AI Cockpit — 预置工作流模板集

## 文章流水线
模式: pipeline
步骤: 大纲 → 初稿 → 审核润色

## 代码审核循环
模式: review_loop
步骤: 开发者写代码 → 审核者检查 → 不通过则循环

## 多角度分析
模式: roundtable
步骤: 数据分析师 → 战略顾问 → 批判性思考者

## AI 辩论赛
模式: debate
步骤: 正方 → 反方 → 评委总结

## 翻译校对
模式: pipeline
步骤: 翻译 → 校对润色

## 短视频脚本
模式: pipeline
步骤: 选题策划 → 脚本撰写 → 口播优化

## 产品需求文档
模式: review_loop
步骤: 需求撰写 → 需求评审 → 修改完善

## 营销文案
模式: roundtable
步骤: 文案策划 → 用户视角 → 数据分析师
